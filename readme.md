# 🌿 Plant Disease Classifier

An end-to-end Deep Learning application that classifies plant leaf images into their respective health categories and diseases using a Convolutional Neural Network (CNN) trained on the PlantVillage dataset. The project includes a Jupyter/Colab training pipeline and a user-friendly Streamlit web interface.

---

## 🚀 Features
* **Automated Pipeline**: Downloads and extracts the PlantVillage dataset directly via the Kaggle API.
* **Custom CNN Architecture**: Implements a high-accuracy Convolutional Neural Network built with TensorFlow/Keras.
* **Interactive Web App**: A lightweight, responsive Streamlit dashboard for real-time image uploads and predictions.
* **Serialized Inference**: Saves model states and class labels using Pickle for instant deployment.

## 🏋️ Model Training (Google Colab)

The model training pipeline is contained within `plantdiseaseclassifier.ipynb`. 

### Dataset Setup
The project utilizes the **PlantVillage Dataset** fetched from Kaggle. To run the training pipeline:
1. Place your `kaggle.json` API token file in your environment.
2. Run the notebook cells to automatically pull, extract, and split the data into 80% Training and 20% Validation sets.

### Model Architecture
* **Input Layer**: Preprocessed RGB images resized to $224 \times 224 \times 3$.
* **Feature Extraction**: 2 Convolutional Layers (`32` and `64` filters) paired with Max Pooling.
* **Classifier**: Fully Connected Dense Layer (`256` units, ReLU) leading to a Softmax output layer matching the total number of dataset classes.
* **Optimization**: Compiled with the `Adam` optimizer and `Categorical Crossentropy` loss.

## 💻 Web App Deployment (Streamlit)

To launch the web interface locally, ensure `model.pkl` and `class_indices.pkl` are present in your working directory, then execute:

```bash
streamlit run app.py
```

### How to Use the App:
1. Open the local URL generated by Streamlit (usually `http://localhost:8501`).
2. Upload a clear image of a plant leaf (`.jpg`, `.jpeg`, or `.png`).
3. The app will visually display your uploaded image and output the predicted health status/disease label in real-time.
